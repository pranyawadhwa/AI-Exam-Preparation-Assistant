# AI Exam Preparation Assistant

An AI-powered study companion that helps students learn from their own notes. Upload PDFs, ask questions, generate summaries, practice questions, MCQs, and memory tricks—all from your study material using Retrieval-Augmented Generation (RAG).

## Project Overview

The AI Exam Preparation Assistant is designed to simplify exam preparation by allowing students to interact with their notes using Artificial Intelligence.

Instead of manually searching through lengthy PDFs, students can upload their notes once and instantly generate revision material tailored to their own content.

---

## Features

- Upload multiple PDF notes
- AI-powered Question Answering
- Study Summary Generation
- Important Question Generation
- MCQ Generation
- Memory Trick Generation
- Semantic Search using Vector Embeddings
- Response Workspace
- Study History
- PDF Export
- Multi-document Support
- Performance Optimizations with Caching

## Tech Stack

| Technology | Purpose |
|------------|---------|
| Python | Backend Development |
| Streamlit | Web Interface |
| Google Gemini API | AI Response Generation |
| Sentence Transformers | Text Embeddings |
| ChromaDB | Vector Database |
| PyMuPDF | PDF Text Extraction |
| ReportLab | PDF Export |

---

## Project Workflow

```text
Upload PDFs
      ↓
Extract Text
      ↓
Generate Embeddings
      ↓
Store in ChromaDB
      ↓
Retrieve Relevant Context
      ↓
Generate AI Response
      ↓
Display & Export Results
```
